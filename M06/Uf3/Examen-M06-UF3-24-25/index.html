Drag and drop

Drag an drop és un event especial que ens permet capturar els events que succceixen durant l'arrosegament d'un element. els events que succeixen són:

  • dragstart
  • dragover
  • dragenter
  • dragleave
  • dragend
  • drop
Entre els event dragstart i drop es pot compartir informació fent servir el metode setData i getData. 
function dragstart(e){
    e.setData('key','value')
}
funcion drop(e){
    let value = e.get data('key', 'value')
}
Per poder permetre l'acció de drop hem de fer e.preventDefault() durant el dragover.

Establir un delay: setTimeout(() => { console.log("Delayed for 1 second."); }, 1000)

Validació Nom i cognoms(2,5 punts)

Verifica que el camp nom i cognoms:
  1. (1p) Ha de contenir com a mínim 2 paraules
  2. (0.5p) Només no pot contenir números
  3. (0.5p) No pot contenir cap dels següents símbols especials ('!','"','·','$','%','&','/','(',')','=','?','¿','*','_','-').
  4. (0.25p) Cada paraula comença amb majúscula i la resta en minsúcules.

Marca com amb la classe valid les validacions que compleixen de la llista. Aquesta validació ha de ser dinàmica, es a dir mentre l'usuari escriu el nom i s'han d'anar marcant els elements de la llista al moment.

(0.25p) Si tot és correcte habilita el botó amb l'id send si es modifica amb dades no valides torna a deshabilitar-ho.

No es pot fer servir expressions regulars per validar el contingut. Els mètodes de natius de javascript si estan permesos.
  • Mínim 2 paraules.
  • No conté cap numero.
  • No conté cap simbol.
  • Cada paraula comença amb majuscula.

Fusió de Pokémons(2.5p)

(1.5 punts) Fes que l'usuari pugui arrossegar els diferents pokémons les diferents àrees per fusionar els pokémons. Quan l'usuari deixi anar un pokémon sobre l'àrea d'un altre fes que els pokémons es fusionin.

Per fusionar els pokémons s'ha de mirar l'atribut data-src que indica el id del pokemon original i l'atribut data-dst que indica el pokémon destí. Si volem fusionar els pokémons 2 i 3 s'ha de modificar el valor de background-image a l'àrea de destí amb el següent valor url('img/pokemonfuse/23.png').

(1 punt) Fes que al posar el pokémon d'origen a sobre del pokémon de destí ja es mostri la fusió al destí, però si el mou a un altre lloc que es torni a mostrar el pokémon original.
