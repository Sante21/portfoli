Drag and drop

Drag an drop és un event especial que ens permet capturar els events que succceixen durant l'arrosegament d'un element. els events que succeixen són:

  • dragstart
  • dragover
  • dragenter
  • dragleave
  • dragend
  • drop
Entre els event dragstart i drop es pot compartir informació fent servir el metode setData i getData. 
function dragstart(e){
    e.setData('key','value')
}
funcion drop(e){
    let value = e.get data('key', 'value')
}
Per poder permetre l'acció de drop hem de fer e.preventDefault() durant el dragover.

Joc de memòria (1,5 punts)

El joc de memòria consisteix en unes cartes on hi ha una parella de cartes amb el mateix dibuix. Aquestes cartes es posen cap per baix i en cada torn el jugador ha de girar dues cartes si aquestes tenen el mateix dibuix les cartes es queden girades. Si les cartes no són iguals quan l'usuari torni a fer clic a una altra carta torna a girar-les per amagar-les. Fes servi la classe show per mostar la carta.
