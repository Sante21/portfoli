Sopa de lletres (3 punts)

La sopa de lletres és un passatemps que consisteix a localitzar una sèrie de paraules en una graella plena de lletres, de manera que es pugui llegir en vertical o horitzontal.

Donat un array amb les paraules que podrien sortir el qual anomenarem diccionari Crea una funció per trobar les paraules dins d'una matriu plena de lletres. Per fer-ho has d'escriure 2 funcions:

  1. cercaHoritzontal(arrayTauler, diccionari): retorna una array amb les paraules trobades en horitzontal.
  2. cercaVertical(arrayTauler, diccionari): retorna una array amb les paraules trobades en vertical.
Si les funcions no troben res retornen null.

No pots fer servir funcions predefinides de cerca per els array, has de fer bucles per trobar les paraules.
 Exemple: 
let diccionari = ['NEU','GORRO','NIT','SOPA','GUANTS','MITJONS',

Validació d'un correu (2 punts)

S'ha de comprovar que el text que hi ha en el input 'passwordEx3' sigui un correu. Les condicions perquè sigui un correu valid són:

  1. Caràcters valids [a-z], [0-9], '-', '_' i '.'.
  2. No es permet el punt al principi ni el final.
  3. Els caràcters - _ . han d'anar seguits per una lletra.
  4. Ha de tenir només una '@'.
  5. Devant i derrere de la '@' ha de tenir text.
  6. Després de '@' ha de tenir text un '.' i més text.

Si no compleix alguna d'aquestes condicions indica les condiccions que falten dins del div 'resultatEx4'.

No pots fer servir funcions de cerca al text només pots repassar el text fent un bucle.
